前端使用vue3
订单管理系统原型系统

功能描述：
（1）客户管理：
①客户入网申请；
②客户入网审批功能；
③客户查询。 
客户申请属性包括：客户名称、客户简称、行业、客户类型、联系人、 联系电话、客户地址等基本属性。 行业、客户类型是选择框后台数据库直接维护即可。
①	客户入网申请； 新增申请、修改、删除、提交、撤销。可以使用选择框
②	客户入网审批功能； 审批或驳回时，备注必须要填写
③	客户查询：可以实现按照客户名称、客户简称、行业、客户类型、联系人、 联系电话、客户地址多个条件查询


（2）合同管理：
①合同录入
②合同审批功能
合同录入属性包括：合同编号、合同名称、签订日期、交货日期、合同 类型、合同金额、客户名称等基本属性、合同编号系统生成，生成规则 根据 合同类型，比如合同类型为产品销售合同，以 C 打头，那么合同编号就是 C202412001 ，格式 = 前缀（1 位） + 年（4 位） + 月（2 位） + 流水 号（3 位）。 
合同录入包括新增、修改、删除、提交、撤销。选择客户是弹出新页面 进行选择，在选择客户页面中，单选一个客户后，再返回到录入页面。合同类型也是选择框的方式
合同类型其数据维护功能不用开发，后台数据库直接维护即可。
合同审批包括驳回、审核、撤销。 
开票管理： 建立发票记录，录入本次开票的合同号、开票金额，本次开票金额需小 于等于合同金额。累计合同的开票金额不能超过合同的总金额。
欠票查询： 该查询主要可以查询合同的已开票金额和未开票金额
